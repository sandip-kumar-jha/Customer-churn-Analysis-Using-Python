# 📊 Customer Churn Analysis 

This project focuses on analyzing customer churn data to identify patterns and factors that lead to customer attrition. The goal is to help businesses improve customer retention using data-driven insights.

---

## 📌 Objective:-

- Analyze customer behavior
  
- Identify key factors responsible for churn
  
- Provide insights to reduce customer loss
  
- Support business decision-making

---

## 📂 Dataset:-

- The dataset contains customer information such as:
  
  - Gender
    
  - Age
    
  - Tenure
    
  - Services subscribed
    
  - Monthly charges
    
  - Churn status (Yes/No)

---

## 🛠️ Tools & Technologies:-

- Python 🐍
  
- Pandas
  
- NumPy
  
- Matplotlib
  
- Seaborn
  
- Google Collab

---

## 🔍 Analysis Performed

- Data Cleaning & Preprocessing
  
- Exploratory Data Analysis (EDA)

- Handling Missing Values
  
- Data Visualization
  
- Correlation Analysis

---

## 📊 Key Insights

- Customers with higher monthly charges are more likely to churn
  
- Short tenure customers have higher churn rate
  
- Certain services impact churn significantly
  
- Contract type plays a major role in retention
